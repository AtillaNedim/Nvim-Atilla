# Nvim-Atilla

Personal Neovim configuration based on Kickstart.nvim.

## Features
- Lazy.nvim plugin manager
- Neo-tree file explorer
- Telescope fuzzy finder
- Floating terminal (ToggleTerm)
- Alt (Meta) based workflow
- Custom keymaps and UI tweaks

## Installation

```bash
git clone https://github.com/AtillaNedim/Nvim-Atilla.git ~/.config/nvim
nvim
